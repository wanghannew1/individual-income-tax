# 中国个税累计预扣计算器

基于 **累计预扣法**（国税局标准算法）的 Web 版个人所得税计算工具。

## 功能特性

- ✅ **累计预扣法标准计算**：严格按国家税务总局公式实现
- ✅ **完整扣除项支持**：
  - 基本减除费用：5,000 元/月
  - 专项扣除：三险一金（养老/医疗/失业/公积金）
  - 7 项专项附加扣除：子女教育、继续教育、大病医疗、住房贷款利息/住房租金、赡养老人、3 岁以下婴幼儿照护
- ✅ **12 个月灵活输入**：可按月填写不同工资/扣除项
- ✅ **逐月明细表**：显示累计收入、累计扣除、累计应纳税所得额、税率、速算扣除数、累计应纳税额、本期应缴
- ✅ **税率跳档高亮**：自动标记税率发生变化的月份
- ✅ **年度汇总卡片**：全年总收入、全年缴税合计、全年税后收入、实际税负率
- ✅ **新旧方法对比**：一键对比“累计预扣法” vs “旧按月计算法”
- ✅ **响应式设计**：桌面端 / 移动端自适应

## 技术栈

- **后端**：Python 3.12 + FastAPI + Jinja2
- **前端**：原生 HTML/CSS（无框架依赖）
- **测试**：pytest（10 个测试用例，覆盖所有税率档位与边界）

## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/wanghannew1/individual-income-tax.git
cd individual-income-tax

# 2. 创建并激活虚拟环境（使用 uv，推荐）
uv venv
source .venv/bin/activate      # Linux/macOS
# .venv\Scripts\activate      # Windows

# 3. 安装依赖（使用清华镜像加速）
uv pip install -r requirements.txt --index-url https://pypi.tuna.tsinghua.edu.cn/simple

# 4. 运行测试（可选）
pytest tests/ -v

# 5. 启动服务
python -m web.app

# 6. 打开浏览器
# http://localhost:8000
```

## 算法说明

**累计预扣法公式**：
```
累计应纳税所得额 = 累计收入 - 累计减除费用(5000×月数) - 累计专项扣除 - 累计专项附加扣除 - 累计其他扣除
累计应纳税额 = 累计应纳税所得额 × 税率 - 速算扣除数
本期应纳税额 = 累计应纳税额 - 已预缴税额
本期应纳税额 ≤ 0 时取 0（暂不退税）
```

**7 级累进税率表**：
| 级数 | 全年应纳税所得额 | 税率 | 速算扣除数 |
|------|------------------|------|------------|
| 1    | ≤ 36,000         | 3%   | 0          |
| 2    | 36,000–144,000   | 10%  | 2,520      |
| 3    | 144,000–300,000  | 20%  | 16,920     |
| 4    | 300,000–420,000  | 25%  | 31,920     |
| 5    | 420,000–660,000  | 30%  | 52,920     |
| 6    | 660,000–960,000  | 35%  | 85,920     |
| 7    | > 960,000        | 45%  | 181,920    |
