# 中国个税累计预扣计算工具

## TL;DR

> **Quick Summary**: 基于 Python (FastAPI) 构建一个 Web 版个税累计预扣计算工具，用户输入每月工资和各项扣除，自动计算每月应纳税额并展示完整计算过程。
>
> **Deliverables**:
> - 核心计算引擎（Python 模块）
> - Web 界面（FastAPI + Jinja2 模板）
> - 完整交互式月收入与扣除输入表单
> - 逐月计算结果展示表（含税率跳档提示）
> - 全年汇总统计
>
> **Estimated Effort**: Short（约 2-3 天工作量）
> **Parallel Execution**: YES - 2 waves + final verification
> **Critical Path**: 计算引擎 → Web 路由 → 前端模板

---

## Context

### Original Request
用户要做一个个税计算工具，基于中国个税累计预扣法，计算工资薪金所得的个人所得税。

### Interview Summary
**Key Decisions**:
- **功能范围**：仅工资薪金累计预扣，不含年终奖/劳务报酬/汇算清缴
- **工具形态**：Web 网页工具
- **技术栈**：Python（FastAPI + Jinja2 模板）
- **子代理策略**：全程禁止使用子代理，所有工作由主代理直接完成

---

## Work Objectives

### Core Objective
构建一个 Web 版个税累计预扣计算器，用户输入每月工资和各项扣除项，系统自动按累计预扣法计算每月应纳税额。

### Concrete Deliverables
- `tax_calculator/core.py` - 核心计算引擎
- `tax_calculator/models.py` - 数据模型定义
- `tax_calculator/tax_brackets.py` - 税率表与专项扣除标准
- `tax_calculator/__init__.py` - 包初始化
- `web/app.py` - FastAPI 应用
- `web/templates/index.html` - 主页面模板
- `web/templates/results.html` - 结果页面模板
- `web/static/style.css` - 样式文件
- `requirements.txt` - 依赖清单
- `pyproject.toml` - 项目配置

### Definition of Done
- [ ] 浏览器打开后看到输入表单页面
- [ ] 填入 1-12 个月工资和扣除项后点击计算
- [ ] 正确显示每月的应纳税所得额、税率、应纳税额、税后收入
- [ ] 计算结果与国家税务总局累计预扣法公式一致
- [ ] 输入非法数据时有友好的错误提示

### Must Have
- 支持 1-12 个月任意月数输入
- 支持每月分别输入工资、三险一金、专项附加扣除
- 7 项专项附加扣除完整支持
- 自动税率匹配 + 税率跳档提示
- 逐月明细表 + 全年汇总
- 累计预扣法和旧按月计算法可切换对比

### Must NOT Have (Guardrails)
- 不要数据库存储（纯无状态计算）
- 不要用户登录/认证系统
- 不要年终奖单独计税功能
- 不要劳务报酬计算
- 不要经营所得计算
- 不要汇算清缴功能

- [x] 2. 核心计算引擎

  **What to do**:
  - 编写 `tax_calculator/core.py`，实现以下函数：
    - `calc_monthly_tax(monthly_data: list)` → list of monthly results
      - 每月数据包含：salary, social_insurance, special_deductions 明细
    - `get_tax_rate(taxable_income)` → (rate, quick_deduction)
    - `calc_annual_summary(monthly_results)` → annual summary
  - 核心算法：
    1. 每月累计收入、累计减除费用(5000×月数)、累计专项扣除、累计专项附加扣除
    2. 累计应纳税所得额 = 累计收入 - 累计扣除总额
    3. 累计应纳税额 = 累计应纳税所得额 × 税率 - 速算扣除数
    4. 本期应缴 = 累计应纳税额 - 已预缴税额
    5. 本期应缴为负值时取 0（暂不退税）
  - 编写 `tests/test_core.py`：
    - 测试案例1：月薪11,000，每月三险一金1,700，住房租金1,500 → 全年3%
    - 测试案例2：月薪40,000，每月三险一金6,000，附加扣除3,000 → 2月跳档到10%
    - 测试案例3：月薪15,000/45,000/15,000（波动收入）→ 验证累计预扣法的平滑效果
    - 测试案例4：空输入（0收入）→ 正确处理
    - 测试案例5：极高收入（>96万/年）→ 验证45%税率

  **Must NOT do**:
  - 不要引入 Web 相关逻辑
  - 不要处理负数退税逻辑（负值取0，不做负数处理）
