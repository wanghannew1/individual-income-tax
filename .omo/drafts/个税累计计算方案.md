# 中国个人所得税累计预扣法 — 完整方案

## 一、法律依据与背景

2019年1月1日起，中国实施新《个人所得税法》，对居民个人的工资薪金所得实行**累计预扣法**（cumulative withholding method），取代旧有的按月独立计算方式。

### 核心变化
- **旧方法**：每月独立计算个税，月与月之间互不影响，收入波动大的月份税负畸高
- **新方法**：按年累计计算，每月预扣税额 = 累计应缴税 - 已缴税，全年税负平滑公平

---

## 二、累计预扣法核心公式

### 2.1 基本公式

```
本期应预扣预缴税额 = (累计预扣预缴应纳税所得额 × 预扣率 - 速算扣除数)
                     - 累计减免税额 - 累计已预扣预缴税额
```

### 2.2 累计应纳税所得额

```
累计预扣预缴应纳税所得额 = 累计收入
                         - 累计免税收入
                         - 累计减除费用（5,000元/月 × 任职月数）
                         - 累计专项扣除（三险一金）
                         - 累计专项附加扣除（7项）
                         - 累计依法确定的其他扣除
```

### 2.3 重要规则
- **负值处理**：本期应预扣预缴税额为负值时，**暂不退税**，次年汇算清缴多退少补
- **年度汇算**：次年3月1日至6月30日办理综合所得汇算清缴

## 四、专项扣除详解

### 4.1 专项扣除（三险一金）

| 项目 | 说明 |
|------|------|
| 基本养老保险 | 个人缴费比例 8% |
| 基本医疗保险 | 个人缴费比例 2% |
| 失业保险 | 个人缴费比例 0.5%（各地略有差异） |
| 住房公积金 | 个人缴费比例 5%~12% |

这些是按月从工资中扣除，进入累计专项扣除。

### 4.2 专项附加扣除（7项）

| 项目 | 扣除标准 | 备注 |
|------|---------|------|
| **3岁以下婴幼儿照护** | 2,000元/月/孩 | 父母各扣50%或一方全额 |
| **子女教育** | 2,000元/月/孩 | 从3岁到博士，父母分摊同上 |
| **继续教育** | 400元/月（学历）或 3,600元/年（职业证书） | 学历教育最长48个月 |
| **大病医疗** | 年度限额80,000元 | 医保报销后个人负担超15,000元部分据实扣除，汇算清缴时申报 |
| **住房贷款利息** | 1,000元/月 | 首套房，最长240个月 |
| **住房租金** | 800/1,100/1,500 元/月 | 按城市规模分三档 |
| **赡养老人** | 3,000元/月（独生子女）或分摊（非独生子女每人≤1,500元） | 父母之一年满60周岁 |

**注意**：住房贷款利息和住房租金**不能同时享受**，同一纳税年度只能选一项。

## 五、计算示例

### 示例1：低收入稳定工资

**条件**：月薪 11,000 元，三险一金 1,700 元/月，住房租金扣除 1,500 元/月

| 月份 | 累计收入 | 累计减除费用 | 累计专项扣除 | 累计专项附加扣除 | 累计应纳税所得额 | 税率 | 速算扣除数 | 累计应缴税 | 本期应缴 |
|------|---------|------------|------------|----------------|---------------|------|----------|----------|---------|
| 1月  | 11,000  | 5,000      | 1,700      | 1,500          | 2,800         | 3%   | 0        | 84       | **84**  |
| 2月  | 22,000  | 10,000     | 3,400      | 3,000          | 5,600         | 3%   | 0        | 168      | **84**  |
| 3月  | 33,000  | 15,000     | 5,100      | 4,500          | 8,400         | 3%   | 0        | 252      | **84**  |

→ 全年一直适用3%税率，每月缴税相同。

### 示例2：中高收入

**条件**：月薪 40,000 元，三险一金 6,000 元/月，三项专项附加扣除共 3,000 元/月

| 月份 | 累计应纳税所得额 | 税率 | 速算扣除数 | 累计应缴税 | 已缴税 | 本期应缴 |
|------|---------------|------|----------|----------|-------|---------|
| 1月  | 26,000        | 3%   | 0        | 780      | 0     | **780** |
| 2月  | 52,000        | 10%  | 2,520    | 2,680    | 780   | **1,900** |
| 3月  | 78,000        | 10%  | 2,520    | 5,280    | 2,680 | **2,600** |
| 4月  | 104,000       | 10%  | 2,520    | 7,880    | 5,280 | **2,600** |

→ 2月起税率跳档到10%，后续月缴增加。

### 示例3：收入波动（累计预扣法的优势）

**条件**：1月工资 15,000 元，2月 45,000 元（含年终奖），3月 15,000 元，子女教育1,000 + 房贷1,000 + 赡养老人2,000 = 4,000 元/月，三险一金 3,000 元/月

| 月份 | 累计应纳税所得额 | 税率 | 速算扣除数 | 累计应缴税 | 已缴税 | 本期应缴 |
|------|---------------|------|----------|----------|-------|---------|
| 1月  | 2,800         | 3%   | 0        | 84       | 0     | **84**  |
| 2月  | 35,600        | 3%   | 0        | 1,068    | 84    | **984** |
| 3月  | 38,400        | 10%  | 2,520    | 1,320    | 1,068 | **252** |

→ 注意：2月虽然收入暴增到45,000元，但累计应纳税所得额35,600元仍在3%档，不会出现旧方法下"一个月多交数千元"的情况。

## 六、工具功能设计

基于上述计算方法，个税计算工具应支持以下功能：

### 6.1 核心计算功能

1. **逐月累计计算**
   - 输入：每月工资收入、三险一金、专项附加扣除
   - 输出：每月应纳税额、累计应纳税额、税后收入

6. **专项附加扣除计算器**
   - 根据个人情况计算可享受的扣除额度

### 6.3 数据管理

7. **多人员管理**
   - 支持多个不同收入情况的纳税人

## 七、技术实现建议

### 核心算法（伪代码）

```
function calcMonthlyTax(monthlyData[]) {
    let cumulativeTaxPaid = 0;
    let results = [];

    for (let i = 0; i < monthlyData.length; i++) {
        let m = monthlyData[i];

        // 计算累计值
        cumulativeIncome += m.salary;
        cumulativeBasicDeduction += 5000;
        cumulativeSocialInsurance += m.socialInsurance;
        cumulativeSpecialDeduction += m.specialDeduction;

        // 累计应纳税所得额
        let taxableIncome = cumulativeIncome
                          - cumulativeBasicDeduction
                          - cumulativeSocialInsurance
                          - cumulativeSpecialDeduction;

        // 查税率表
        let [rate, quickDeduction] = getTaxRate(taxableIncome);

        // 累计应纳税额
        let totalTax = taxableIncome * rate - quickDeduction;

        // 本期应缴
        let monthlyTax = totalTax - cumulativeTaxPaid;
        if (monthlyTax < 0) monthlyTax = 0; // 负值暂不退税

        cumulativeTaxPaid = totalTax;
        results.push({ cumulativeTaxableIncome, totalTax, monthlyTax });
    }
    return results;
}
```
